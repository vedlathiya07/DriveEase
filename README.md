# 🚗 DriveEase – MERN Car Rental Platform

## 🔥 Features

- Car booking system
- Payment integration (Razorpay)
- Map + nearby services
- Admin dashboard
- Image upload
- Condition reports

---

## 🛠 Tech Stack

Frontend: React, Bootstrap  
Backend: Node.js, Express  
Database: MongoDB

---

## 🚀 Run Locally

### Backend

cd DRIVEEASE
npm install
node app.js

### Frontend

cd client
npm install
npm run dev

---

## 🌍 Deployment

### Backend (Render)

- Push to GitHub
- Create Web Service
- Add ENV variables

### Frontend (Vercel)

- Import repo
- Set build command: npm run build

---

## 🔐 ENV VARIABLES

DB_URL=
JWT_SECRET=
RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=
